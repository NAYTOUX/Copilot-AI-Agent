# Decision Records

Decision records capture durable architecture choices for the agent hub. Keep
them short and append-only.

## Format

- `docs/adr/NNNN-title.md`
- Status: `proposed`, `accepted`, `superseded`, or `rejected`.
- Include context, decision, consequences, and validation.
